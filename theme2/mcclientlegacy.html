The not-so famous AND1558 Client (Now Discontinued!)

This contains informations about the AND1558 Minecraft Client that ive been developing since 2021

The client is based of Minecraft, specifically the version 1.8.9

This client is made to be somewhat of a replica of the Hyperium Client
which was discontinued in May 2021, it also contains some code from Hyperium

Updates are rarely made as this is just my passion project.
I do plan on making support for newer Minecraft Versions
ex: 1.9 above
but my Minecraft modding experience stops at <1.9

The Installer

There are 2 versions of the installer:
Cross-Platform Java Installer
Windows-only .NET 6 Installer

Cross-Platform Java Installer has alot of Graphical Bugs and will be fixed in future versions!
